# Eiken Magic Words PWA 安装说明

适用对象：iPhone、iPad、Google Pixel、Android 手机和平板。

安装地址：

```text
https://bobodanpapa.github.io/EikenMagicwordsPWA/
```

PWA 不是从 App Store 或 Google Play 下载的 App。它是一个可以安装到手机桌面的网页 App。安装后，桌面上会出现 `Eiken Magic Words` 图标，点图标就可以像普通 App 一样打开。

## 苹果手机 / iPad

请使用 Safari 安装。不要从微信、LINE、Gmail、Google App 或 Chrome 里面直接安装，因为这些浏览器内置页面可能不会显示正确的安装按钮。

1. 打开 Safari。
2. 在 Safari 地址栏输入：

   ```text
   https://bobodanpapa.github.io/EikenMagicwordsPWA/
   ```

3. 页面打开后，点击 Safari 底部或顶部的分享按钮。
4. 在菜单里找到并点击 `添加到主屏幕` / `Add to Home Screen`。
5. 名字保持 `Eiken Magic Words`，点击 `添加` / `Add`。
6. 回到 iPhone 或 iPad 主屏幕，找到 `Eiken Magic Words` 图标。
7. 以后请从主屏幕图标打开，不要每次从浏览器书签打开。

如果看不到 `添加到主屏幕`：

- 确认正在使用 Safari，不是微信、LINE、Gmail、Google App 或 Chrome 内置浏览器。
- 在分享菜单里往下滑，或点击 `编辑操作`，看看是否能把 `添加到主屏幕` 加回来。
- 确认主屏幕还有空位，或者用系统搜索搜索 `Eiken Magic Words`。
- 复制安装地址，重新用 Safari 打开一次再试。

## Android 手机 / Google Pixel

请优先使用 Chrome 安装。Google Pixel 和大多数 Android 手机的步骤基本一样，不同品牌的按钮名称可能略有不同。

1. 打开 Chrome。
2. 在 Chrome 地址栏输入：

   ```text
   https://bobodanpapa.github.io/EikenMagicwordsPWA/
   ```

3. 页面打开后，点击右上角的三个点菜单。
4. 点击 `Install app`、`安装应用`、`Add to Home screen` 或 `添加到主屏幕`。
5. 如果系统弹出确认窗口，点击 `Install` / `安装` / `添加`。
6. 回到手机桌面或应用列表，找到 `Eiken Magic Words` 图标。
7. 以后请从这个图标打开。

如果页面底部自动弹出安装提示，也可以直接点提示里的 `Install` / `安装`。

如果看不到安装按钮：

- 确认正在使用 Chrome，并且打开的是 HTTPS 地址。
- 先刷新页面，再打开右上角三个点菜单查看。
- 如果只显示 `Add to Home screen`，也可以点击；有些 Android 版本会用这个名字表示安装。
- 如果从微信、LINE、短信或 Gmail 点开的链接，请复制地址后改用 Chrome 打开。
- 如果还是不行，重启 Chrome 或更新 Chrome 后再试。

## 安装后怎么更新

PWA 会通过网页自动更新。通常重新打开 App，或关闭后再打开，就会拿到新版本。

如果朋友说看到的内容还是旧的：

1. 关闭 `Eiken Magic Words`。
2. 重新打开一次。
3. 如果还是旧的，用浏览器打开安装地址刷新一次。
4. 最后仍然不更新时，可以删除桌面图标，再按上面的步骤重新安装。

## 删除方法

苹果手机 / iPad：

1. 长按主屏幕上的 `Eiken Magic Words` 图标。
2. 点击 `移除 App` 或 `删除书签`。
3. 确认删除。

Android：

1. 长按 `Eiken Magic Words` 图标。
2. 点击 `卸载` / `Uninstall`，或拖到 `卸载` 区域。
3. 确认删除。

## 注意事项

- 安装和第一次打开需要联网。
- 学习进度保存在当前手机和当前浏览器里，不会自动同步到其他手机。
- 如果换手机、换浏览器、删除浏览器数据，学习进度可能不会一起带过去。
- iPhone 请用 Safari 安装；Android 请优先用 Chrome 安装。
- 如果麦克风或发音功能不能用，请确认浏览器允许麦克风权限，并且页面是通过 HTTPS 地址打开的。
